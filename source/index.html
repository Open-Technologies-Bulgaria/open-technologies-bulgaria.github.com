Voyager

Voyager is a simple software distribution platform for Linux. It provides access control for yum repositories so that only authorized customers are able to download and install the software. The platform consists of a yum plugin, Drupal interface to manage accounts and subscriptions and a CGI process installed on the HTTP server. We're using Voyager to provide add-on software to our clients.
